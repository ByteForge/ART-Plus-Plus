ART++.com
=========

Description
-----------

An upcoming website with apps. The main concept behind the desing is to use the 3 base colors (actually the 3 close CMYK colors without the black) through the whole website including the logo itself.

Developed With
--------------
* IntelliJ PHPStorm
* IntelliJ WebStorm

Used Techniques
---------------
* jQuery
* SASS/SCSS
* GSAP (TweenLite)
